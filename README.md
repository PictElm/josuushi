## [助数詞](https://jisho.org/word/%E5%8A%A9%E6%95%B0%E8%A9%9E)「じょすうし」

Build with:
```console
$ elm make Main.elm --output=main.js
```

The result is a static site:
- `index.html`
- `style.css`
- `main.js`
- `data/`

Example:
```console
$ serve &
```
